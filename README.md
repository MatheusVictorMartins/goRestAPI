# Go Rest API
<p>Apenas um pequeno projeto realizado durante estudos da linguagem de programação Go! Com o objetivo de aprender a subir uma RestAPI com PostgresSQL e bibliotecas da linguagem como:</p>
<ul>
<li>Gorilla Mux</li>
<li>Gorilla Handlers</li>
<li>GORM</li>
</ul>
<p>Além disso, foi também reforçado o consumo de APIs com front-end básico em React.js</p>
